📱 Exportador HTML para PNG Mobile

Ferramenta para converter código HTML em imagens PNG de alta fidelidade, formatadas para mobile (400px).

✨ Funcionalidades

  • Motor SVG: Renderização perfeita de fontes e ícones (dom-to-image).
  • Qualidade Retina: Imagens exportadas com densidade 2x.
  • Fatiamento Manual: Corte imagens longas com precisão.
  • ZIP Automático: Baixe fatias agrupadas em um arquivo .zip.

🚀 Como Usar

  1. Cole seu HTML na aba Editor.
  2. Vá para a aba Preview.
  3. Escolha Baixar PNG para imagem única.
  4. Escolha Cortes para fatiar manualmente:
    • Toque na imagem para criar linhas de corte.
    • Use o "X" para remover uma linha.
    • Clique em Baixar ZIP.
